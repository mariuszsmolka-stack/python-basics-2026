# Instrukcja warunkowa if w Pythonie

## Cel lekcji

Poznasz instrukcję warunkową `if` i nauczysz się wykonywać wybrany fragment kodu tylko wtedy, gdy warunek jest spełniony.

## Czego się nauczysz

Po tej lekcji będziesz umieć zapisać prosty warunek w Pythonie. Nauczysz się używać dwukropka po warunku, robić poprawne wcięcia oraz sprawdzać proste warunki z użyciem operatorów porównania.

## Po co używamy instrukcji if

Instrukcji `if` używamy wtedy, gdy program ma coś sprawdzić i wykonać kod tylko w określonej sytuacji.

`if` pozwala wykonać kod tylko wtedy, gdy warunek jest spełniony, czyli gdy wynik warunku ma wartość `True`.

Przykład: program może sprawdzić, czy liczba jest dodatnia. Jeśli jest dodatnia, wyświetli komunikat. Jeśli nie jest dodatnia, ten komunikat się nie pojawi.

## Składnia instrukcji if

Podstawowa składnia wygląda tak:

```python
if warunek:
    instrukcja
```

Po warunku stawiamy dwukropek `:`.

Instrukcje należące do `if` muszą być wcięte. Najczęściej używamy czterech spacji.

Kod wcięty wykona się tylko wtedy, gdy warunek jest prawdziwy.

Kod bez wcięcia nie należy już do `if`.

## Przykład: liczba dodatnia

```python
liczba = int(input("Podaj liczbę: "))

if liczba > 0:
    print("Liczba jest dodatnia.")

print("Koniec programu.")
```

Omówienie wiersz po wierszu:

Pierwszy wiersz wczytuje liczbę od użytkownika i zamienia ją na liczbę całkowitą.

Wiersz z `if` sprawdza warunek `liczba > 0`.

Jeśli warunek jest prawdziwy, program wypisuje komunikat `Liczba jest dodatnia.`.

Napis `Koniec programu.` wyświetli się zawsze, bo nie jest wcięty i nie należy do instrukcji `if`.

## Przykład: sprawdzanie wieku użytkownika

```python
wiek = int(input("Podaj swój wiek: "))

if wiek >= 18:
    print("Jesteś osobą pełnoletnią.")
```

Program wczytuje wiek użytkownika. Następnie sprawdza, czy wartość zmiennej `wiek` jest większa lub równa `18`. Jeśli tak, wyświetla komunikat.

## Przykład: temperatura większa od 30

```python
temperatura = float(input("Podaj temperaturę: "))

if temperatura > 30:
    print("Jest bardzo ciepło.")
```

Program wczytuje temperaturę jako liczbę rzeczywistą. Jeśli temperatura jest większa niż `30`, pojawia się komunikat.

## Przykład: hasło

```python
haslo = input("Podaj hasło: ")

if haslo == "python":
    print("Hasło poprawne.")
```

Program wczytuje hasło jako tekst. Operator `==` sprawdza, czy wpisany tekst jest taki sam jak `"python"`.

## Operator porównania ==

Operator `==` służy do sprawdzania, czy dwie wartości są równe.

```python
liczba = int(input("Podaj liczbę: "))

if liczba == 10:
    print("Podano liczbę 10.")
```

W tym przykładzie komunikat pojawi się tylko wtedy, gdy użytkownik wpisze `10`.

## Operator porównania !=

Operator `!=` służy do sprawdzania, czy dwie wartości są różne.

```python
imie = input("Podaj imię: ")

if imie != "":
    print("Witaj,", imie)
```

Program sprawdza, czy użytkownik wpisał jakikolwiek tekst. Jeśli zmienna `imie` nie jest pusta, program wyświetla powitanie.

## Typowe błędy uczniów

### Brak dwukropka po warunku

```python
liczba = int(input("Podaj liczbę: "))

if liczba > 0
    print("Liczba jest dodatnia.")
```

Po warunku musi być dwukropek `:`.

### Brak wcięcia pod instrukcją if

```python
liczba = int(input("Podaj liczbę: "))

if liczba > 0:
print("Liczba jest dodatnia.")
```

Instrukcja należąca do `if` musi być wcięta. Bez wcięcia Python zgłosi błąd.

### Mylenie = oraz ==

```python
haslo = input("Podaj hasło: ")

if haslo = "python":
    print("Hasło poprawne.")
```

Znak `=` służy do przypisania wartości do zmiennej. Do porównania używamy `==`.

Poprawnie:

```python
haslo = input("Podaj hasło: ")

if haslo == "python":
    print("Hasło poprawne.")
```

### Porównanie liczby bez konwersji

```python
wiek = input("Podaj swój wiek: ")

if wiek >= 18:
    print("Jesteś osobą pełnoletnią.")
```

Funkcja `input()` zwraca tekst. Jeśli chcemy porównać wiek z liczbą, trzeba użyć `int()` albo `float()`.

Poprawnie:

```python
wiek = int(input("Podaj swój wiek: "))

if wiek >= 18:
    print("Jesteś osobą pełnoletnią.")
```

### Myślenie, że print() bez wcięcia należy do if

```python
liczba = int(input("Podaj liczbę: "))

if liczba > 0:
    print("Liczba jest dodatnia.")

print("Ten napis wyświetli się zawsze.")
```

Ostatni `print()` nie jest wcięty, więc nie należy do instrukcji `if`. Wykona się zawsze.
