# Python od podstaw 2026

Repozytorium zawiera materiały do kursu Python od podstaw dla klasy I liceum.

Celem kursu jest spokojne wprowadzenie uczniów w podstawy programowania:
pisanie pierwszych programów, korzystanie ze zmiennych, pobieranie danych od
użytkownika, stosowanie warunków i prostych pętli oraz wykonanie małych
projektów.

Materiały kursu znajdują się w katalogu [docs](docs/index.md).
